Restful Routes

A table of 7 Restful Routes

Name Path HTTP Verb Purpose
Index /blogs GET List of all blogs
New /blogs/new GET Show new blog form
Create /blogs POST Create a new blog, then redirect somewhere
Show /blogs/:id GET Show info about one specific blog
Edit /blogs/:id/edit GET Show edit form for one blog
Update /blogs/:id PUT Update a particular blog, then the redirect somewhere
Destroy /blogs/:id DELETE Delete a particular blog, then the redirect somewhere
